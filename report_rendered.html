1. Arsitektur Komponen

Dua instance MTCNN digunakan karena perannya berbeda:
• mtcnn_single (keep_all=False) → foto referensi/selfie: cukup 1 wajah confidence tertinggi
• mtcnn_all (keep_all=True) → foto yang di-index: semua wajah diambil agar foto ramai bisa ditemukan

2. Pencarian Foto — POST /search

📐 Rumus Cosine Similarity

sim(A, B) = (A · B) / (|A| × |B|)

A = embedding query vektor 512D dari selfie
B = embedding dari database vektor 512D

Hasil: 0.0 (tidak mirip) → 1.0 (identik)
Threshold default: 0.40 (diatur di .env SIMILARITY_THRESHOLD)
Deduplication: Satu foto N orang → N baris di DB. Hanya similarity tertinggi per drive_file_id yang disimpan: 
if fid not in best_per_file or sim > best_per_file[fid]["similarity"]:
    best_per_file[fid] = { "similarity": sim, ... }

3. Pengelompokan Foto — POST /cluster

📐 Parameter & Rumus DBSCAN

cosine_distance(A, B) = 1 - cosine_similarity(A, B)

eps = 0.4       → dua wajah "orang sama" jika cosine_sim >= 0.6
min_samples = 2 → minimal 2 wajah membentuk cluster (1 wajah = noise label -1)

Pemilihan foto representatif:
  centroid = mean(embeddings_norm[members], axis=0)
  rep_idx  = argmax(cosine_similarity(centroid, cluster_embs))

4. Klasifikasi Objek — POST /analyze

📐 Rumus Weighted Human Score

human_score = skin_score   × 0.35    (Skin Color HSV Analysis)
            + texture_score × 0.30    (Laplacian Variance)
            + color_score   × 0.20    (Color Std Dev RGB)
            + conf_score    × 0.15    (MTCNN Confidence)

Normalisasi ke 0–1:
  skin_score    = 1.0 jika skin_ratio 0.20–0.75, linear di luar rentang
  texture_score = min(1.0, laplacian_var / 250.0)
  color_score   = 1.0 jika color_std 12–60, linear di luar rentang
  conf_score    = min(1.0, mtcnn_confidence)

5. Indexing Multi-Face

Kenapa Multi-Face Penting? 
Sebelum fix (keep_all=False):  Foto 5 orang → 1 embedding → 4 orang TIDAK ditemukan
Sesudah fix (keep_all=True):   Foto 5 orang → 5 embedding (face_index 0,1,2,3,4)

UNIQUE (drive_file_id, face_index) → ON DUPLICATE KEY UPDATE → re-index aman
